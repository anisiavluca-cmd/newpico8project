# newpico8project
The game is a simple 2D platformer set in a small medieval-inspired world. The player controls a young messenger who has been sent to recover stolen royal seals that were taken from the kingdom.

Each level represents a different location, such as castle walls, forest paths, or underground passages. The messenger must jump across platforms, avoid traps like spikes or pits, and navigate simple obstacles to reach the stolen item at the end of each level.

The character’s goal is to retrieve all the seals and return them to the castle, restoring order in the kingdom.
Simple controls (left, right, slide and jump).
